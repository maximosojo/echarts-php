Echarts-PHP
========================

EChartsPHP es una librería PHP para la libreria de gráficas **Echarts js** (https://github.com/ecomfe/echarts). Soportar Echarts en las versiones 2.2.x to 4.x.

## Instalación

``` bash
$ composer require maximosojo/echarts-php
```
